## What's Left To Do

### Icons — REQUIRED (manual step)
The PWA and iOS home screen need PNG icon files. These cannot be generated by code without an image tool.

You need to create the following files in `public/icons/`:

| File | Size | Purpose |
|------|------|---------|
| `icon-192.png` | 192×192 | Android home screen / PWA |
| `icon-512.png` | 512×512 | Android splash screen / PWA |
| `icon-maskable-192.png` | 192×192 | Android adaptive icon (content in centre 80%) |
| `icon-maskable-512.png` | 512×512 | Android adaptive icon (content in centre 80%) |
| `apple-touch-icon.png` | 180×180 | iOS home screen icon |

**How to generate them:**

Option A — Online tool (easiest):
1. Go to https://realfavicongenerator.net or https://www.pwabuilder.com/imageGenerator
2. Upload `public/icons/icon-source.svg`
3. Download the icon pack and drop the files into `public/icons/`

Option B — Use ImageMagick locally:
```bash
magick public/icons/icon-source.svg -resize 192x192 public/icons/icon-192.png
magick public/icons/icon-source.svg -resize 512x512 public/icons/icon-512.png
magick public/icons/icon-source.svg -resize 180x180 public/icons/apple-touch-icon.png
# maskable icons are the same PNGs (the SVG already has padding built in)
cp public/icons/icon-192.png public/icons/icon-maskable-192.png
cp public/icons/icon-512.png public/icons/icon-maskable-512.png
```

## How to Test

1. Add the PNG icon files (step above)
2. Deploy to `hub.retrocyclesleague.com`
3. On iPhone: open in Safari → Share button → "Add to Home Screen"
4. On Android: open in Chrome → the install banner will appear automatically
5. Launch from home screen — opens with no browser chrome, RCL icon visible
